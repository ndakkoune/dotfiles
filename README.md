# Dotfiles


## Installation

To install the dotfiles, run the following commands:

```
git clone git@github.com:ndakkoune/dotfiles.git --recursive 
cd dotfiles && ./install
```

## Requirements

- git
- tmux
- [nvim](https://github.com/neovim/neovim)
- [lazygit](https://github.com/jesseduffield/lazygit)
